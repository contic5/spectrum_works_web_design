CSS

CSS stands for Cascading Style Sheets and is used for styling website elements.

CSS can modify the appearance for element types, element classes and element ids.

Some of the attributes you can change for an element include:

  1. Font Size
  2. Color
  3. Background Color
  4. Font
  5. Positioning
  6. Text Alignment
  7. Font Weight

JavaScript

Javascript is used to program the front end behavior of a webpage. You can create dynamic and modifiable webpages with JavaScript

Many other popular web development frameworks use Javascript, so it is a very important language to learn.

JavaScript can change the following key areas

  1. HTML Content
  2. HTML Attribute Values
  3. HTML Styles
  4. Visibility of HTML Elements

JavaScript can

  1. Handle form submission
  2. Continuously update components
  3. Create new elements
  4. Use conditions to handle program flow
  5. Use loops to repeat code
  6. Use functions to call code from different places

Vanilla JavaScript cannot

  1. Get serverside files
  2. Get values from a database
  3. Save data

jQuery

jQuery is a Javascript library. jQuery code is just easier to write and more efficient JavaScript code. jQuery helps with “HTML DOM (Document Object Model) Manipulation, Event Handling, Animation and Ajax,” according to W3Schools.

jQuery get element by ID
myElement=document.getElementById(“id01”);
myElement = $("#id01");

jQuery get element value
myValue=document.getElementById(“id01”).value;
myValue=$(“#id01”).val();

PHP

PHP is used to program the backend behavior of a webpage

With PHP, you can get data from a server.

While there are alternatives to PHP, it is still relatively simple to learn and quite productive

PHP can do the following

  1. Handle Server Data
    1. Create Data
    2. Read Data
    3. Update Data
    4. Delete Data
  2. Save data in a session variable
  3. Handle form submissions.
  4. Create new elements
  5. Use conditions to handle program flow
  6. Use loops to repeat code
  7. Use functions to call code from different places

Vanilla PHP cannot

  1. Continuously update components
  2. Overwrite existing elements

Ajax

AJAX stands for Asynchronous JavaScript and XML. It uses JavaScript and XML to access web servers from a web page in real time rather only when the page first loads. Ajax can also use jQuery.

AJAX allows for the following useful features

  1. Handle Server Data in real time rather than only when the page first loads
    1. Create Data without reloading the page
    2. Read Data after the page has loaded
    3. Update Data without reloading the page
    4. Delete Data without reloading the page

AJAX can get regular text files. Ajax can also access PHP files that can run and then return a result, which allows for getting and handling server data with PHP.

Data can also be passed to AJAX functions.
